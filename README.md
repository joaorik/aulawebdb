# Aula Web DB

Feito em JSP

Faça o import do banco de dados:
```sh
aula07.sql
```

Configure a conexão de seu host em:
```sh
src\java\web\aula\db\MySqlConnect.java
```
 
Acesse a home, clique em login para criar o primeiro usuario.

Pronto, trabalho finalizado.
